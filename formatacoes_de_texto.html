Citações

A tag 'q' gera citações simples no texto (com semântica).
como a seguinte: O Computador é um burro muito rápido.

Para citações longas (completas) usamos a tag 'blockquote' como abaixo:

      

      Lorem ipsum dolor sit amet, consectetur adipisicing elit,
